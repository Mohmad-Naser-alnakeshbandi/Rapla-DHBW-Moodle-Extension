## To use this extension: </br>
1- you need to download the entire folder </br>
2- unzip it </br>
3- use Chrome browser</br>
4- open extensions panel</br>
5- enable Developer mode</br>
6- click ok load unpacked </br>
7- select the unziped file</br>
